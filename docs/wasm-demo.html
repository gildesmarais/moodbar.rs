MOODBAR.RS / WASM DEMO

Analyze audio in-browser and render a moodbar SVG.

This demo uses the Rust analysis core compiled to WebAssembly. Pipeline: decode with Web Audio API, mix to mono PCM, run analyze(), render with svg(). Supported module locations: ../packages/moodbar-wasm/ (local repo root) and ./assets/moodbar-wasm/ (GitHub Pages). Audio is analyzed at 11,025 Hz mono for better responsiveness on large files.
